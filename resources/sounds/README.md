# Звуки

AudioCache (см. `src/audio/AudioCache.h`) пытается лениво загрузить
файлы по этим путям. Если файла нет — игра пишет один варнинг и
просто молчит. Никаких крашей.

Дропни сюда `.wav` (или `.ogg`) с соответствующими именами и они
заиграют сами без правок кода.

| Файл              | Когда играет                                 |
|-------------------|----------------------------------------------|
| shot.wav          | Любой выстрел игрока                          |
| hit.wav           | Пуля попала во врага                          |
| enemy_death.wav   | Обычный враг умер                             |
| player_hurt.wav   | Игрок получил урон (контакт или пуля)         |
| level_up.wav      | Поднялся уровень (перед UpgradeScene)         |
| pickup.wav        | Поднял пикап (оружие)                         |
| boss_warn.wav     | Появился босс (конец волны)                   |
| boss_death.wav    | Босс убит                                     |

Имена констант: см. `src/Paths.h` → `namespace paths::sfx`.

Бесплатные источники: https://freesound.org/, https://opengameart.org/,
https://sfxr.me/ (генератор 8-bit звуков).
